1. 用户管理

1.1. 用户注册

  • Method：POST

  • Path: /register

  • Body 参数

参数 类型 必填 默认值 描述
name String true 无 用户名
password String true 无 密码
phone String true 无 手机号
email String true 无 邮箱
introduce String false '' 个人介绍
type String false 1 注册类型

  • type 0：博主，1：其他用户 ，2：github， 3：weixin， 4：qq ( 0，1 是注册的用户； 2，3，4 都是第三方授权登录的用户)

  • 示例

{
  "name": "coco",
  "password: "246",
  "phone": "136786786",
  "email": "admin@163.com",
  "instroduce": "nihao",
  "type": 1
}
  • 成功返回结果示例
{
  "code": 0,
　"message":null,
　"data":{}
}
  • 失败返回结果示例
{
  "code": 1,
  "message": "系统异常,请稍后重试",
  "data": {}
}

keyword String true '' 关键字
content String true '' 内容
